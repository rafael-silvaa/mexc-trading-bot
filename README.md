# MEXC Futures Trading Bot

Trading automation middleware focused on the MEXC Futures market. The system receives Webhook signals generated by custom logic in TradingView (Pine Script) and asynchronously executes orders on the exchange.


## System Architecture

1. **TradingView (The Brain):** Executes the *Pine Script*, validates conditions (e.g., moving average crossovers, volatility filters), and triggers real-time alerts.
2. **Python Backend (The Bridge):** Server built with Flask that listens for *POST requests* (Webhooks). It parses the JSON payload, applies risk management rules, and interacts with the exchange API.
3. **MEXC API (The Execution):** Receives the buy/sell instructions, strictly operating in **Isolated Margin** mode.



## Prerequisites

* **Python:** 3.9 or higher.
* **TradingView:** Essential plan (or higher) to enable Webhook delivery.
* **MEXC:** API Keys (`API_KEY` and `SECRET_KEY`) enabled for the Futures market (starting with Mock/Testnet environment keys is highly recommended).
* Remote server or VPS for 24/7 deployment.
